# Sets

Create and perform ***very basic binary operations*** on sets within the shell.

## Quickstart
```shell
$ make run
```

## Creating And Changing Sets
```
-> x = {...}
-> x = {1, 2, 3}
-> x = {3, 2, 1}
```

## Binary Operators
```
-> a union b
-> a intersect b
-> a compliment b
-> a subset b
```

## Print Sets
```
-> a = {1, 2, 3}
-> a
       {1, 2, 3}
```

## Number Of Elems
```
-> x = {1, 2, 3}
-> |x|
   3
```
